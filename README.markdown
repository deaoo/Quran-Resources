	    <u><b>COND1,COND2 :</b></u>
	        - for 'ayas' :
	                        index of aya like: 37
	                        OR sura-aya like: 2-10
	        - for 'suras' :
	                        index of sura like: 113 
	    
	    <u><b>Note :</b></u> COND2 is optional ..
